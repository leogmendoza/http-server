### Setup Instructions
1. Clone my repo:
   ```bash
   git clone https://github.com/leogmen/http-server.git
   cd http-server
2. Create build directory:
   ```bash
   mkdir build
   cd build
3. Configure project with CMake:
   ```bash
   cmake ..
5. Build executable:
   ```bash
   cmake --build .
6. Run the server!:
   ```bash
   ./http_server
7. In your browser, visit: [http://localhost:8080/](http://localhost:8080/)
